# Basic Skill 系列导读

在 AI 时代，实操不再重要，理解原理才是关键。掌握复杂的计算机系统工程，首先需要理解计算机是如何运作的。

本系列教程旨在帮助你理解计算机是如何运作的，以及如何使用它。我们不会涉及过分复杂的底层原理，而是专注于从系统构建的角度，为你快速构建对于计算机的直觉认识。
