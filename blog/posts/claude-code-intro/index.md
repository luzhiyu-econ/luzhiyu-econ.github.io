# Claude Code 系列导读

## 核心原则

> 对于 Vibe Coding 的方法论，最重要的是**先计划再实施，先思考再动手**。这不仅是对模型的要求，也是对使用者的要求。

> **分步骤编码**：一小块功能一小块功能地进行实现，如果需要最后对接，再让它单独处理对接的问题。不要寄希望于它能够完全基于自然语言理解和遵循你的想法和预期。
