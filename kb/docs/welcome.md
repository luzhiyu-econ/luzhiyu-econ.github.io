# Welcome to My Knowledge Base

这是我的个人知识库，用于记录科研过程中的学习笔记、方法论心得和文献整理。

## 如何使用

- **左侧目录树** — 按文件夹浏览所有文档
- **#标签过滤** — 点击顶部标签筛选相关文档，或在搜索框输入 `#标签名`
- **关键词搜索** — 在搜索框中输入关键词快速定位文档

## 标签体系

本知识库使用自定义标签系统来组织内容：

| 标签 | 含义 |
|------|------|
| `#public-finance` | 公共财政相关 |
| `#nlp` | 自然语言处理 |
| `#methodology` | 研究方法论 |
| `#literature` | 文献阅读笔记 |

> 本知识库仅供个人使用，不在主页公开链接。
