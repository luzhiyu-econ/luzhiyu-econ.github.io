# Welcome to My Knowledge Base

> 这是我的个人知识库，用于记录科研过程中的学习笔记、方法论心得和文献整理。


## 标签体系

本知识库通过层级标签组织内容，在 frontmatter 中使用 `#父级/子级` 格式的标签自动归入对应分类：

| 标签前缀       | 内容     |
| ------------ | ------ |
| `research/*` | 研究经验总结 |
| `skills/*`   | 各类教程   |

常用子标签：`skills/tools`、`skills/nlp`、`research/academic` 等。一篇文章可拥有多个标签，会在对应分类下同时出现。
