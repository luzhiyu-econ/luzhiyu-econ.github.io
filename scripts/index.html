zhiyu-lu.com  ·  Claude Code 通用安装脚本

macOS Linux install-claude.sh bash
交互式（推荐）
$ bash -c "$(curl -fsSL https://zhiyu-lu.com/scripts/install-claude.sh)"
静默式（预设环境变量）
$ CLAUDE_TOKEN="sk-xxx" CLAUDE_API_URL="https://your-api.com" bash -c "$(curl -fsSL https://zhiyu-lu.com/scripts/install-claude.sh)"
运行后依次提示输入 API Token 和 API Base URL，两项均为必填。
安装流程：Node.js 检查 → Bun → Claude Code（官方脚本优先，失败回退 Bun）→ 写入 ~/.claude/settings.json。
Windows install-claude-win.ps1 PowerShell
交互式（推荐）
PS> irm https://zhiyu-lu.com/scripts/install-claude-win.ps1 | iex
静默式（预设环境变量）
PS> $env:CLAUDE_CLIENT_TOKEN='sk-xxx'; $env:CLAUDE_API_URL='https://your-api.com'; irm https://zhiyu-lu.com/scripts/install-claude-win.ps1 | iex
需要 PowerShell 3.0+。若提示执行策略限制，先运行：
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

安装流程：Node.js 检查（winget 优先）→ Bun → Claude Code（官方脚本优先，失败回退 Bun）→ 写入 %USERPROFILE%\.claude\settings.json。
