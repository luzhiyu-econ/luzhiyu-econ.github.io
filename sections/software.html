pybdiff

v0.1.0

Python replication of Stata's bdiff command. Tests whether regression coefficients differ across two subgroups via permutation tests, bootstrap tests, or parametric Wald tests — with optional parallel execution.

pip install pybdiff
Quick Example
# Test whether x1 and x2 coefficients differ between group 0 and 1
from pybdiff import bdiff

result = bdiff(
    df      = df,
    group   = "treated",
    formula = "y ~ x1 + x2 | firm + year",
    vcov    = {"CRV1": "firm"},
    method  = "permutation",   # or "bootstrap" / "wald"
    reps    = 500,
    n_jobs  = -1,              # parallel: use all cores
)

pyshapley2

v0.1.0

Python replication of Stata's shapley2 command. Computes the Shapley-Owen decomposition of any regression fit statistic (R², adjusted R², log-likelihood, AIC, …) across independent variables or user-defined variable groups, with optional parallel computation.

pip install pyshapley2
Quick Example
# Decompose R² across independent variables
from pyshapley2 import shapley2

result = shapley2(
    df       = df,
    depvar   = "wage",
    indepvars = ["edu", "exp", "tenure"],
    stat     = "r2",        # or "r2_a", "ll", "aic", "bic"
    n_jobs   = -1,          # parallel: use all cores
)
result.summary()            # Stata-style table output

pybacondecomp

v0.1.0

Python replication of Stata's bacondecomp command. Implements the Goodman-Bacon (2021) decomposition of any TWFE DiD estimator into all 2×2 comparisons, supporting uncontrolled, ddetail, and FWL-controlled branches with optional multi-core parallelism.

pip install pybacondecomp
Quick Example
# Decompose TWFE DiD into all 2×2 comparisons
from pybacondecomp import bacondecomp, bacon_plot

result = bacondecomp(
    df    = df,
    y     = "outcome",
    tr    = "treat",        # binary, weakly increasing
    unit  = "state",
    time  = "year",
    n_jobs = -1,            # parallel: use all cores
)
bacon_plot(result)          # scatter: estimates vs. weights
